# IPL-Matches-EDA
EDA on IPL matches from 2008-2024
Exploratory Data Analysis (EDA) of Indian Premier League (IPL) matches from 2008 to 2024.

## Objective
To analyze match-level trends and patterns in IPL matches, focusing on team performance,
toss decisions, match outcomes, and seasonal trends.

## Key Analyses
- Top teams by number of wins
- Impact of toss on match outcomes
- Batting first vs bowling first analysis
- Seasonal trend of matches played
- Venue-wise win margin distribution (runs vs wickets)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Public IPL match dataset covering seasons from 2008 to 2024.
